# 🧪 ATX Research Peptides
**Premium Headless Shopify Storefront**

A high-performance e-commerce platform for research-grade compounds, built with Next.js and Shopify Storefront API.

---

### 🌟 Key Features
- **Headless Shopify**: Real-time product & cart management.
- **Secure Checkout**: Authentication guard for safe payments.
- **Order Tracking**: Custom live tracking interface.
- **Lab Animations**: Dynamic Lottie-based scientific visuals.
- **Premium UI**: Dark-themed glassmorphism design.

---

### 🛠️ Tech Stack
- **Frontend**: Next.js 15 (App Router)
- **E-Commerce**: Shopify Storefront API
- **Motion**: Lottie-React
- **State**: React Context API

---

### 🚀 Quick Start
```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

### 🔑 Environment Variables
Create `.env.local`:
```env
NEXT_PUBLIC_SHOPIFY_STORE_DOMAIN=your-store.myshopify.com
NEXT_PUBLIC_SHOPIFY_STOREFRONT_ACCESS_TOKEN=your_token
```
